# AnnotationCheckbox

**Properties**

| Name    | Type                             | Required | Description                     |
| :------ | :------------------------------- | :------- | :------------------------------ |
| Checked | bool                             | ❌       | Whether the checkbox is checked |
| Style   | signplus.AnnotationCheckboxStyle | ❌       | Style of the checkbox           |
